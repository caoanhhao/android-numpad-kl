Android numpad keyboard layout is based on TitanQwerty and provides custom keyboard layouts

What to expect:
- Use physical keyboard numpad number

Use instructions: 
1. Open app
2. Tap on "physical keyboard settings"
3. Tap on top entry (your physical keyboard)
4. Tap on "set up keyboard layouts"
5. Scroll down and select desired Layout (Full keyboard with numpad, US QWERTY)
6. Go back and select the layout you just added as active layout
